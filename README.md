# Program do gry w kółko i krzyżyk

Skrypt jest gotowym programem do gry z Chatem GPT w kółko i krzyżyk. Program wykorzystuje domyślnie wersję modelu gpt-4.1-mini.
Można zmienić wersję modelu w klasie GPTPlayer w zmiennej "model_name". Do skorzystania z programu wymagany jest plik typu .env,
który zawiera własny 'OPENAI_API_KEY'.
